```

太懒了不想写 Commit message，

@cursor_ai

 帮你写👇：

1⃣ 按住 [Control ⌃ + `] 打开终端输入 git add 暂存更改。

2⃣ 按住 [Command ⌘ + K] 对话框输入 "git commit message" 然后提交请求，搞定！简洁明了。

```
